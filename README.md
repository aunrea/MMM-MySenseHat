# MMM-MySenseHat
Magic Mirror Raspberry Pi Sense Hat Module

# Installation
* clone repository
* within folder, npm install

# Installation errors
* check your /home/pi/.pm2/log/
* if NODE COMPILING VERSION ERROR
  ./node_modules/.bin/electron-rebuild
